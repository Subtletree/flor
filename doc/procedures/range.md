
# range

"range" is a procedure to generate ranges of integers.

```
# range {end}
# range {start} {end}
# range {start} {end} {step}
range 0       #--> []
range 4       #--> [ 0, 1, 2, 3 ]
range 4 7     #--> [ 4, 5, 6 ]
range 4 14 2  #--> [ 4, 6, 8, 10, 12 ]
```
